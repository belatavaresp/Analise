Podemos ter qualquer resultado escolhendo sequências diferentes e tomando o [[Limite]]. Essas indeterminações podem ser obtidas afrouxando as restrições das [[Limite infinito#Propriedades]].

1. $(+\infty) - (+\infty)$
2. $0\cdot \infty$
3. $\frac{0}{0}$
4. $\frac{\infty}{\infty}$

**Outras:**
- $O^{\infty}$
- $1^{\infty}, (1+\frac{1}{n})^n\to e$
- $0^0$
